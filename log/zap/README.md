# zap

基于 [zap](https://github.com/uber-go/zap) 开发的日志包，致力于提升 zap 使用体验。

## 文档

[Go 第三方 log 库之 zap 使用](https://jianghushinian.cn/2023/03/19/use-of-zap-in-go-third-party-log-library/)

## 特性

- [x] 类似 log 标准库的 API 设计
- [x] 动态修改日志级别
- [x] 可以设置不同日志级别输出到不同位置
- [x] 日志轮转，支持按时间/日志大小
